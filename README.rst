Flask-Triangle
--------------

**With great UX comes great security needs.**
*Uncle Ben*


Implements a Flask-WTF look-alike extension to create forms with AngularJS
interactions.


Features :

    * easy binding beetween a form and a JSON object
    * client and server side validation

Requirements :

    * server side :

      - Flask (of-course)
      - jsonschema
      - six

    * client side :

      - AngularJS
